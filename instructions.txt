## 2. Dataset Setup & Image Normalization (`utils/dataset_utils.py`)

1. **Automatic Download**: 
   - If the directory `./data/tiny-imagenet-200/` is missing or empty, download the official dataset archive programmatically:
     `http://cs231n.stanford.edu/tiny-imagenet-200.zip`
   - Extract it into the `data/` directory.
2. **Explicit Constants**:
   - Define exact mean and standard deviation values for the RGB channels of Tiny ImageNet.
   - You must use explicit constants for transformations:
     - `RED_MEAN`, `GREEN_MEAN`, `BLUE_MEAN`
     - `RED_STD`, `GREEN_STD`, `BLUE_STD`
   - Use these constants to construct standard `transforms.Normalize` and its inverse transformation for plotting.

## 3. Model Wrappers & Weights Management (`models/`)

1. **Wrapper Class**:
   - Create a PyTorch `nn.Module` wrapper class or custom class that contains the underlying model.
   - The wrapper must expose a `.name` property.
   - The name must dynamically adapt based on:
     - The architecture type (e.g., `resnet18`, `efficientnet_v2_s`).
     - Whether it loaded pretrained weights (e.g., `_pretrained` or `_scratch`).
2. **Standardized Saving Format**:
   - All weights must be saved in the `weights/` directory.
   - File naming pattern: `<model_name>_<training_type>.pth`
     - `<model_name>`: Extracted from `<model>.name` (e.g., `resnet18_pretrained`).
     - `<training_type>`: Must be one of `clean`, `labels` (for training using teacher's hard labels), or `logits` (for soft distillation using KL Divergence).

## 4. Attack Taxonomy & Implementation (`attacks/evasion_attacks.py`)

1. **Nomenclature Update**:

2. **Function Decoupling**:
   - Isolate FGSM and PGD implementations.

## 5. Jupyter Demo Notebook (`demo.ipynb`)

1. Create a clean demonstration notebook.
2. Run evaluations using varying perturbation budgets (epsilon values, e.g., `[0.0, 0.01, 0.03, 0.05, 0.1]`).
3. Generate and display line plots comparing:
   - Target model standard accuracy.
   - White-box attack accuracy (FGSM and PGD directly against the target).
